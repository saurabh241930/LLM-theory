Union Find: The Disjoint Set Union

Connectivity Tracking, Path Compression, and Inverse Ackermann Complexity

1. Technical Deep-Dive: Tracking Groups

Union Find (or DSU) is a data structure designed to solve the "Dynamic Connectivity" problem: Given two elements, are they in the same group? If not, can we merge their groups?

The Dual Optimizations: Path & Rank

A naive Union Find is O(N). To reach near-O(1), we apply two clever tricks:

  • Path Compression: During the `find` operation, we make every node on the path point directly to the root. This flattens the tree.
  • Union by Rank/Size: We always attach the smaller tree under the larger tree. This keeps the height minimal.

Complexity: With both, the operations take O(α(N)) time, where α is the Inverse Ackermann function. For any N < 10600, α(N) < 5. Effectively O(1).

Key Use Cases

  • Network Connectivity: Are two computers in a network linked?
  • Cycle Detection: Helping Kruskal's Algorithm find the Minimum Spanning Tree.
  • Image Processing: Segmenting pixels into distinct connected components.

A. The Robust UnionFind Class

A reusable boilerplate for most connectivity problems.

JavaScript
Python
class UnionFind {
    constructor(n) {
        this.parent = Array.from({length: n}, (_, i) => i);
        this.rank = new Array(n).fill(0);
    }

    find(i) {
        if (this.parent[i] === i) return i;
        // Path Compression
        return this.parent[i] = this.find(this.parent[i]);
    }

    union(i, j) {
        let rootI = this.find(i);
        let rootJ = this.find(j);
        if (rootI !== rootJ) {
            // Union by Rank
            if (this.rank[rootI] < this.rank[rootJ]) {
                this.parent[rootI] = rootJ;
            } else if (this.rank[rootI] > this.rank[rootJ]) {
                this.parent[rootJ] = rootI;
            } else {
                this.parent[rootI] = rootJ;
                this.rank[rootJ]++;
            }
            return true;
        }
        return false;
    }
}
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n

    def find(self, i):
        if self.parent[i] == i:
            return i
        # Path Compression
        self.parent[i] = self.find(self.parent[i])
        return self.parent[i]

    def union(self, i, j):
        root_i = self.find(i)
        root_j = self.find(j)
        if root_i != root_j:
            # Union by Rank
            if self.rank[root_i] < self.rank[root_j]:
                self.parent[root_i] = root_j
            elif self.rank[root_i] > self.rank[root_j]:
                self.parent[root_j] = root_i
            else:
                self.parent[root_i] = root_j
                self.rank[root_j] += 1
            return True
        return False

B. Kruskal's Algorithm (Minimum Spanning Tree)

Sorting edges and using DSU to add them if they don't form a cycle.

JavaScript
Python
function minSpanningTree(edges, n) {
    // 1. Sort edges by weight
    edges.sort((a, b) => a.weight - b.weight);
    let uf = new UnionFind(n);
    let totalWeight = 0;

    for (let {u, v, weight} of edges) {
        // 2. Add edge only if it connects two different sets
        if (uf.union(u, v)) {
            totalWeight += weight;
        }
    }
    return totalWeight;
}
def kruskals(edges, n):
    # Sort by weight O(E log E)
    edges.sort(key=lambda x: x[2])
    uf = UnionFind(n)
    mst_weight = 0
    
    for u, v, weight in edges:
        # Union Find effectively prevents cycles
        if uf.union(u, v):
            mst_weight += weight
            
    return mst_weight
