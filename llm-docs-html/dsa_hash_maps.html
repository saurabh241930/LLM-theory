The Power of O(1) Search Patterns

1. Two Sum (The Gateway Problem)

Intuition

The "Brute Force" approach involves two nested loops to check every possible pair, which is O(n²). To improve this, we must trade space for time.

Goal: Given nums and a target, find i and j such that nums[i] + nums[j] == target.

Hash Map Strategy: As we iterate through the array, we ask: "Have we seen a number X such that X + current_num == target?"

  • Calculate the complement = target - current_num.
  • Lookup complement in our Map. If it exists, we found our pair!
  • If not, add current_num to our Map (with its index as the value) and move on.
Time Complexity: O(n) Space Complexity: O(n)
