Self vs Cross Attention

Where do the Queries, Keys, and Values actually come from?

The Mathematics of Source

Every Attention mechanism universally computes Softmax(Q · K) · V. The mathematical equation is completely identical in all architectures. The only difference between Self-Attention and Cross-Attention is physical plumbing: where the data originates before it enters the matrices.

Self-Attention: In a standard Transformer layer (like GPT), a sequence only talks to itself. A single embedding vector ("The cat sat") mathematically forks into 3 separate identical copies. Then, linear weights project them into Q, K, and V. The words query against themselves, find keys within themselves, and extract values from themselves.
