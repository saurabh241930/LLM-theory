Convolutional Neural Networks (CNN)

Extracting spatial features via sliding matrix windows

The Mathematics of Vision

If you feed a 1000x1000 pixel image into a standard Feed-Forward Network, the first layer would need 1 Million inputs. The weights would instantly explode to billions, and the model would completely fail to understand that a "cat ear" in the top-left is the same thing as a "cat ear" in the bottom-right.

Convolutions solve both problems. Instead of looking at every pixel at once, CNNs drag tiny (e.g., 3x3) mathematical filters (kernels) across the image. The filter multiplies overlapping pixels and sums them into a single number.

Step-by-step feature extraction:

1. The Yellow Box (3x3) sits over the input grid.

2. We multiply the 9 pixels inside the box by the exactly 9 corresponding weights in the Kernel Filter.

3. We instantly sum up those 9 multiplications into a single number: the Green output feature. By dragging this same exact 3x3 filter across the whole image, we achieve Translation Invariance—if the filter detects a vertical line, it will detect it no matter where it is on the grid!
