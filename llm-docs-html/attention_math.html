The Geometry of Attention

We often write the Attention equation as a matrix multiplication, but linear algebra is just geometry in disguise.

When the Transformer multiplies a Query Matrix by a Key Matrix, it is fundamentally computing the Dot Product between two vectors. Geometrically, the Dot Product is calculating "How much of the Query vector points in the exact same direction as the Key vector?"

The Dot Product mathematically shrinks or grows Value (V).

The "Aha!" Moment: The Yellow Dashed Line represents projecting the Query directly onto the Key. The length of this shadow is the Dot Product.

Notice the Green Value (V) Vector sticking out on the right.
The Value vector is the actual payload containing the semantic meaning of the token. The Transformer takes the raw number from the Dot Product (how aligned Q and K are) and multiplies it against the V vector.

If you drag Q to point exactly opposite of K, the Dot Product becomes negative/zero, and the Green V vector shrinks to nothing. The network completely ignores it!
