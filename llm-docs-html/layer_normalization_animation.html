Layer Normalization

Taming Exploding Numbers and Recovering Lost Freedom

As data passes through extremely deep layers in a Transformer, repeated multiplications cause "Internal Covariate Shift"—the scale of the numbers gets wildly distorted. Gradients explode or vanish. Layer Normalization forcibly clamps every token's feature vector back into a stable, identical scale.
