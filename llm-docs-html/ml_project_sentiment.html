1. Project Description

Understanding public opinion is vital for brand management. In this project, we analyze a stream of social media comments to determine if they are positive, negative, or neutral. We then visualize this in a dashboard.

Workflow: Scrape Data -> Clean Text -> Sentiment Scoring -> Visualize.

3. Text Processing

We remove stop words, punctuation, and emojis to prepare the text for the VADER sentiment engine.

In [2]:
def get_sentiment(text):
    score = sia.polarity_scores(text)
    if score['compound'] >= 0.05: return 'Positive'
    elif score['compound'] <= -0.05: return 'Negative'
    else: return 'Neutral'

df = pd.read_csv('social_media_feed.csv')
df['Sentiment'] = df['text'].apply(get_sentiment)
