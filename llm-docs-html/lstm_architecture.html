Long Short-Term Memory (LSTM)

Solving the Vanishing Gradient with the Cell State Conveyor Belt

The Mathematics of Gates

An RNN overwrites its memory entirely at every time step, mathematically crushing old signals. An LSTM solves this by splitting its brain into two channels: The Hidden State (Short-term) and the Cell State (Long-term).

The Cell State acts as a frictionless super-highway that gradients can flow across without vanishing. The LSTM protects this highway using Sigmoid Gates (Values exactly between 0.0 and 1.0) that act as physical valves to let information in, forget information, or read it out.

Slide the Gates to see the physical math metaphor:

  • If the Forget Gate is 0.0, the valve completely pinches the Cell State pipe shut, mathematically wiping the long-term memory to 0.
  • If the Input Gate is 1.0, the valve fully opens, allowing the new word (x_t) to completely flood into and update the Cell State.
  • If the Output Gate is 0.0, the Cell State still remembers everything internally, but refuses to output it to the user at this specific time step.
