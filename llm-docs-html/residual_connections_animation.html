Residual Connections

Solving the Vanishing Gradient via Gradient Highways

In ultra-deep networks (like 100-layer Transformers), backpropagation signals (gradients) physically shrink as they are repeatedly multiplied through the layers. If they reach 0, early layers stop learning entirely! We fix this with Residual Skip Connections.
