Feed-Forward Network (FFN)

The factual memory bank of the Transformer

The Unsung Hero of the Transformer

While the Attention Mechanism gets all the glory (it figures out "who is talking to whom" in a sentence), standard Transformers spend 2/3 of their parameters in the Feed-Forward Network (FFN). If Attention is the routing system, the FFN is the database where the model stores its knowledge about the real world.

Every single token passes through its own identical FFN independently. The math is simple: a linear up-projection, an activation function, and a linear down-projection.

The Mathematics of the FFN

1. The 4x Expansion Ratio

The standard Transformer FFN projects the embedding dimension (say, 4096) into a massive hidden space (16384 dimensions) before squeezing it back down to 4096.

Why? Capacity. The network learns billions of specific combinations of concepts in this massive intermediate space. Research suggests that the intermediate neurons in the FFN act as "key-value memories." The first layer acts as a pattern matcher (Key), and the second layer outputs the associated concept (Value).

2. The Activation Function (GeLU / SwiGLU)

The original Transformer used a simple ReLU activation. Modern LLMs (like LLaMA 3) use SwiGLU (Swish-Gated Linear Unit).

Unlike ReLU which simply kills negative numbers, SwiGLU uses a smooth curve that allows tiny negative values to pass through. More importantly, it uses a gating mechanism: one fully connected layer acts as the standard data vector, while a second parallel layer acts as a "gate" (0 to 1) deciding exactly how much of that data vector should be allowed to progress.

SwiGLU(x) = (xW1) ⊗ Swish(xW2)

3. Storing Factual Knowledge

Recent research ("Transformer Feed-Forward Layers Are Key-Value Memories") proved that FFNs memorize facts. If you prompt GPT with "The capital of France is", the Attention mechanism attends to "capital" and "France". The combined embedding is then shoved into the FFN.

The FFN's first layer recognizes the pattern for [Country_Capital_Query: France]. This perfectly activates neuron #12044. The second layer projects neuron #12044 outwards into the exact vector representation of "Paris".
