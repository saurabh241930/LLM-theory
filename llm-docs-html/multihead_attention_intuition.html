Why Multi-Head Attention?

The Problem With Averaging Everything

Multi-Head Attention is the core reason Transformers are so powerful. Instead of using a single "Context Searcher" (which would result in a messy, blurred average of grammatical and semantic relationships), the network splits its searching power across multiple independent heads.
