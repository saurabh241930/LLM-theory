1. Remove Duplicates from Sorted Array

Explanation & Intuition

If we could use extra memory (O(n)), we could easily solve this using a Hash Set. However, given that the input is Sorted and we must use O(1) extra space, we can exploit the ordering.

Key Observation: Since it's sorted, duplicates always appear consecutively. Once we "skip" a group of identical numbers, they will never reappear.

Fast/Slow Strategy:

  • The Slow pointer tracks the position of the last unique element confirmed.
  • The Fast pointer scans ahead.
  • When arr[fast] != arr[slow], we've found a new unique number. We increment slow and write the new value there.

Time Complexity: O(n) Space Complexity: O(1)

PSEUDO-CODE (PYTHON STYLE)
slow = 0
for fast in range(1, len(nums)):
if nums[fast] != nums[slow]:
slow += 1
nums[slow] = nums[fast]
return slow + 1
