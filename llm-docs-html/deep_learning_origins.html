Origins of Deep Learning

From the Perceptron to the Multi-Layer Perceptron (MLP)

1. The Perceptron (1950s)

The atom of deep learning. It mimics a biological neuron: multiple inputs are weighted, summed, and passed through an activation function.

graph LR X1[Input x1] -- w1 --> S((Σ)) X2[Input x2] -- w2 --> S B[Bias b] --> S S --> F[Activation Function] F --> Y[Output y]

Limitation: Single perceptrons can only solve "Linearly Separable" problems (They fail at XOR logic!).

2. Multi-Layer Perceptron (The "Deep" in DL)

By stacking layers of perceptrons, we create a Neural Network. The layers between input and output are called "Hidden Layers." This allows models to learn Non-linear relationships.

graph LR I[Input Layer] --> H1[Hidden Layer 1] H1 --> H2[Hidden Layer 2] H2 --> O[Output Layer]

3. Activation Functions: Adding Non-Linearity

Without activation functions, a 100-layer neural network is mathematically the same as a 1-layer model. They allow the network to "bend" the space.

ReLU (Rectified Linear Unit)

f(x) = max(0, x)

Pros: Computationally efficient, avoids the Vanishing Gradient Problem in most cases. Most common in LLM feed-forward layers.

Cons: "Dying ReLU" – neurons can output 0 and never recover if weights aren't updated. Fix: Leaky ReLU or SwiGLU.

Sigmoid vs. Softmax

  • Sigmoid: Output [0,1]. Used for binary classification. Prone to vanishing gradients in deep networks.
  • Softmax: Multi-class output. Sum of all probabilities equals 1. Used in the final layer of LLMs for token prediction.

4. Normalization: Stabilizing the Network

Normalization ensures that values flowing through the network stay within a reasonable range, speeding up training.

Technique Mechanism Used in...
Batch Norm Normalizes across the Batch dimension. CNNs, Vision models.
Layer Norm Normalizes across the Feature dimension. Transformers (LLMs). Works better for variable-length sequences.

5. The Vanishing Gradient Problem

During backpropagation, gradients are multiplied layer by layer. If gradients are small (using Sigmoid/Tanh), they shrink exponentially, and the early layers 'never learn'.

Solution: Use ReLU activation and Residual Connections (Skip connections).

6. Backpropagation & Universal Approximation

The Universal Approximation Theorem states that a neural network with even a single hidden layer can approximate any continuous function. Backpropagation is the algorithm that propagates the error backward through the network to update the weights.
