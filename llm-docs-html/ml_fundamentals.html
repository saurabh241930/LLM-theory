1. Supervised Learning: The Starting Point

Before jumping into LLMs, we must understand the core of prediction: Supervised Learning. This is where a model learns a mapping from inputs (X) to outputs (y) based on labeled examples.

2. Linear Regression (Predicting Numbers)

Linear regression is the simplest algorithm, often used for predicting continuous values (e.g., house prices, sales revenue).

ŷ = β₀ + β₁x₁ + β₂x₂ + ... + ε

In our RSA Sales QA case study, linear regression could be used to predict the length of a call based on the complexity of the inquiry.

3. Logistic Regression (Predicting Classes)

Despite the name, this is a Classification algorithm. It predicts the probability that an input belongs to a certain category (0 or 1).

The Sigmoid Function: Maps any real value into a range between 0 and 1.
σ(z) = 1 / (1 + e⁻ᶻ)

Use Case: Predicting if a sale will be "Successful" or "Unsuccessful" based on historical customer interaction data.

4. The Bias-Variance Tradeoff

This is the most critical concept in all of Machine Learning. It explains why models fail to generalize.

Concept Problem State Solution
High Bias Underfitting Too simple, misses patterns Add complexity / features
High Variance Overfitting Too complex, learns noise Regularization / Simplify
