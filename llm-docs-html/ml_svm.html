1. Summary

SVM seeks to find the Hyperplane that best separates two classes with the "maximum margin." The data points closest to this hyperplane are called Support Vectors—they are the critical points that define the boundary.

4. The Kernel Trick

When data is not linearly separable in 2D, SVM uses a Kernel function to project it into a higher-dimensional space where a linear split *is* possible.

5. Real-World Case Study: Handwriting Recognition (OCR)

The Problem: A postal service needs to automatically read zip codes on envelopes to sort mail.

The Solution: SVM classifies pixel grids of handwritten digits (0-9).

  • Features: Intensity of each pixel in an 8x8 grid.
  • Decision: Is this character an '8' or a 'B'?

Why SVM? Hand-drawn characters often have very subtle differences. SVM is excellent at finding the Maximum Margin between similar-looking classes in high-dimensional space (64 pixels), making it highly robust to varying handwriting styles.
