Feature Engineering & Data Preprocessing

Turning Raw Data into Predictive Signals

1. Why Engineering Matters

Algorithms are only as good as the data they receive. In traditional ML, 80% of the work is often preprocessing.

2. Essential Techniques

Data Cleaning

  • Imputation: Filling missing values (Mean, Median, or specialized models).
  • Outlier Removal: Getting rid of erroneous data points that skew the mean.

Categorical Encoding

Models only understand numbers. We must convert text categories into numerical form.

  • One-Hot Encoding: Creating binary columns for each category (City_NY, City_LA).
  • Label Encoding: Assigning a unique integer (NY=0, LA=1).

Numerical Scaling

If one feature is "Age" (0-100) and another is "Income" (0-1,000,000), the model will think Income is 10,000x more important just because the number is larger.

  • Standardization (Z-score): (x - μ) / σ
  • Normalization (Min-Max): Rescaling to 0-1 range.

3. Dimensionality Reduction (PCA)

Principal Component Analysis reduces the number of features while preserving as much variance as possible. This is the 1D, 2D, and 3D plotting predecessor to modern high-dimensional Embeddings.
