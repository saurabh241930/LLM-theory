Recurrent Neural Networks (RNN)

Visualizing Time Steps and the Vanishing Gradient

The Mathematics of Memory

Unlike standard Feed-Forward networks, an RNN attempts to understand time. It processes a sentence word-by-word, maintaining a continual "Hidden State" vector that mathematically merges the current word with all the memories of the past words.

Interactive Unrolled RNN Visualizer

Watch the Hidden State (Green Arrow): Every time you click "Process Next Word", the RNN takes the current input word and merges it with the existing Hidden State using a tanh activation matrix multiply.

The Vanishing Gradient Problem: Notice the red "Gradient Signal" bar at the bottom. During backpropagation, the gradients must travel backwards through time (from T=4 all the way to T=0). Because they are repeatedly multiplied by weights < 1.0, they geometrically shrink to Zero. By step 4, the RNN has completely mathematically "forgotten" the word at T=0. This fatal flaw necessitated the invention of the LSTM.
