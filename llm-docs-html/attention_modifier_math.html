Self-Attention: Intuition & Math

Visualizing Context-Aware Embeddings

Self-Attention calculates a Context-Aware Embedding for each token by computing a weighted sum of all surrounding tokens in the sequence. In the video tutorial approach, we start by understanding linguistically which modifiers apply to a noun, and then prove it mathematically using the Key-Query dot products to generate the exact percentages (weights).
