Linear Algebra in Transformers

Matrix multiplication as physical space transformation

The "3Blue1Brown" Perspective of Attention Matrices

In standard tutorials, you are told that multiplying an Embedding Vector by a Weight Matrix ($W_q$) creates a Query Vector. But what does that physically mean?

Every Weight Matrix in a neural network is a Linear Transformation. It physically grabs the entire geometric grid of space and stretches, shears, and rotates it. The vector representing our token (e.g., "Apple") is just caught in the fabric of this space and moves along with it into a new "Query Space".

Visualizing the Wq Projection

Watch the grid. The yellow arrow represents a 2D Token Embedding. The Green (i-hat) and Red (j-hat) arrows are our basis vectors.

When you click "Apply", we multiply the space by the Query Matrix W_q = [[1.5, -0.5], [0.8, 1.2]].

Notice how the grid lines remain parallel and evenly spaced (the definition of a linear transformation). The Token Embedding geometrically moves to a completely new coordinate—it has been successfully projected into the Query Space, ready to be dot-producted against Keys.
