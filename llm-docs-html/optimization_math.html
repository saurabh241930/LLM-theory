Optimization: The Engine of Learning

Loss Functions, Gradient Descent, and Convergence

1. Loss Functions: Measuring Error

Every ML model needs a way to measure how wrong its predictions are. The goal of optimization is to minimize this "Loss."

  • MSE (Mean Squared Error): Standard for Regression.
  • Cross-Entropy Loss: Standard for Classification and LLMs.

2. Gradient Descent: Walking Downhill

Imagine being on top of a foggy mountain (High Loss) and wanting to find the valley (Minimal Loss). You take steps in the direction of the steepest descent.

graph TD A[Start at Random Weights] --> B[Calculate Gradient/Slope] B --> C[Update Weights: w = w - η * ∇] C --> D{Is Loss Minimal?} D -->|No| B D -->|Yes| E[Model Converged!]

Learning Rate (η): The Step Size

If η is too high, you overstep the valley and diverge. If η is too low, the model takes forever to learn.

3. Modern Optimizers: Beyond SGD

While basic Gradient Descent works, modern high-scale models use more sophisticated algorithms that adapt the learning rate dynamically.

Adam: The Gold Standard

Adam (Adaptive Moment Estimation) effectively combines two ideas: Momentum (v) and RMSProp (s).

m_t = β₁ m_{t-1} + (1 - β₁) g_t
v_t = β₂ v_{t-1} + (1 - β₂) g_t²
ŵ = w - η * (m_t / (√v_t + ε))
  • m_t (1st Moment): Estimates the mean of the gradients (Momentum). Helps dampen oscillations.
  • v_t (2nd Moment): Estimates the uncentered variance (RMSProp). Scales learning rate for each parameter individually.
