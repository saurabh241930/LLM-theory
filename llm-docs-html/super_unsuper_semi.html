Learning Paradigms

Supervised, Unsupervised & Self-Supervised Learning

The Three Pillars of Machine Learning

🎯 Supervised Learning

Goal: Learn a mapping from inputs (X) to labels (Y).

Data: Fully labeled (e.g., thousands of emails manually marked "Spam" or "Not Spam").

  • Classification: Predicting categories.
  • Regression: Predicting numbers.

🌌 Unsupervised Learning

Goal: Discover hidden structures in data without human guidance.

Data: Completely unlabeled.

  • Clustering: K-Means, DBSCAN (Grouping similar customers).
  • Dimensionality Reduction: PCA, t-SNE (Compressing variables).

🌉 Semi & Self-Supervised

Goal: Leverage massive unlabeled data using a small amount of labeled data, or generate labels automatically.

Self-Supervised (LLM Core!): The bridge to modern AI. The data provides its own labels.

  • Next Token Prediction: Hide the next word, predict it, then repeat. (GPT)
  • Masked Language Modeling: Hide middle words, predict them. (BERT)
