The Mathematics of Optimization

Every neural network in existence is just an insanely massive, deeply nested algebraic equation. To adjust the weights, we must figure out the Partial Derivative (how much the final Loss changes if we tweak one specific weight by a tiny amount).

Because the network is nested, we use the Chain Rule of Calculus. We calculate the gradient at the very end (the Loss), and literally push it backwards through the mathematical operations, multiplying local derivatives together.
