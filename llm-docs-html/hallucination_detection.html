Hallucination Detection Beyond RAG

Self-Consistency, Factuality Scoring, and Uncertainty Quantification

The Post-RAG Problem

Even if you perfectly retrieve the correct document via RAG, the LLM can still misinterpret it, ignore it, or confidently invent facts. Detecting hallucination precisely before sending the answer to the user is the Holy Grail of LLMOps.

There are four primary strategies to detect a lie:

1. Self-Consistency (Ensemble Math)

Primarily used for logic or math problems. Instead of asking the model once, you ask the model 5 times (with temperature=0.7) to generate 5 different reasoning paths.

If 4 paths result in "42" and 1 path results in "39", you take the majority vote (42), and assume the outlier was a hallucination. Extremely effective, but costs 5x the API tokens.

2. Factuality & NLI Scoring (LLM-as-a-Judge)

Natural Language Inference (NLI). You deploy a second, smaller model specifically trained to detect contradiction.

You feed the Judge model the RAG source document and the generated Answer. The Judge grades the Answer:
Entailment (True), Neutral (Unsupported), or Contradiction (Lie). This is the foundation of frameworks like RAGAS and TruLens.

3. Uncertainty Quantification (Logprobs)

Models like OpenAI allow you to request the logprobs array alongside the generated text. This exposes the literal probability percentage the model assigned to every word it chose.

If a model generates "The capital of France is Paris [99% prob]", it is confident. If it generates "The patient should take Aspirin [12% prob]", the model is violently hallucinating and guessing the next token. You programmatically catch low-confidence entities and trigger a fallback.

4. SelfCheckGPT (Zero-Resource Detection)

What if you don't have RAG source documents to check against?

Mechanic: You generate 1 "Main Answer" (Temp=0.0). Then you generate 5 "Draft Answers" (Temp=1.0) on the exact same question. You then ask a small LLM to cross-examine passage 1 against the 5 drafts. If the LLM is hallucinating a specific date, it will generate a different wrong date every single time in the 5 drafts. The cross-examination effortlessly spots the contradiction and flags the Main Answer as a hallucination.
