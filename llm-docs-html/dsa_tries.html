Tries: The Prefix Oracle

Retrieval Trees, Autocomplete Engines, and Space Complexity Trade-offs

1. Technical Deep-Dive: Fast String Retrieval

A Trie (from "Retrieval") is an ordered tree-like structure used to store strings. Each path from the root down to a node represents a common prefix.

The Time-Space Paradox

Time: Searching for a word of length L is exactly O(L). This is effectively constant time for human-readable words, regardless of how many million words are in the tree. Much faster than O(L * log N) for a sorted list.

Space: Space is the weakness. Every node holds a map or array of 26 pointers (for English letters). Total space: O(ALPHABET_SIZE * N * L). Solution: Use a Hash Map inside each node instead of an array to save space for sparse nodes.

When to Use?

  • Autocomplete: "Find all words starting with 'cat'".
  • Spell Check: Instant "valid word" lookup.
  • Longest Common Prefix: Finding shared roots efficiently.

A. Standard Trie Implementation

The core building block with `insert`, `search`, and `startsWith` methods.

JavaScript
Python
class TrieNode {
    constructor() {
        this.children = {}; // Map char to TrieNode
        this.isEndOfWord = false;
    }
}

class Trie {
    constructor() { this.root = new TrieNode(); }

    insert(word) {
        let node = this.root;
        for (let char of word) {
            if (!node.children[char]) node.children[char] = new TrieNode();
            node = node.children[char];
        }
        node.isEndOfWord = true;
    }

    search(word) {
        let node = this.root;
        for (let char of word) {
            if (!node.children[char]) return false;
            node = node.children[char];
        }
        return node.isEndOfWord;
    }
}
class TrieNode:
    def __init__(self):
        self.children = {} # {char: TrieNode}
        self.end = False

class Trie:
    def __init__(self):
        self.root = TrieNode()

    def insert(self, word: str):
        node = self.root
        for c in word:
            if c not in node.children:
                node.children[c] = TrieNode()
            node = node.children[c]
        node.end = True

    def starts_with(self, prefix: str) -> bool:
        node = self.root
        for c in prefix:
            if c not in node.children:
                return False
            node = node.children[c]
        return True
