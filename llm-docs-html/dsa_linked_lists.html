Pointer-Based Connectivity and Memory Management

1. Detect Cycle (Floyd's Tortoise and Hare)

Explanation & Intuition

If we could use extra memory (O(n)), we would use a Hash Set to record nodes we've seen. But O(1) space requires a more clever strategy.

The Racing Analogy: Imagine two runners on a track. One (the Hare) runs twice as fast as the other (the Tortoise).

  • If the track is Straight (no cycle), the Hare will simply reach the finish line (NULL) first.
  • If the track is Circular (a cycle), the Hare will eventually "lap" the Tortoise from behind. They will meeting at some point inside the cycle.

Floyd's Algorithm: We use two pointers. If they ever become equal, a cycle exists. If the faster one reaches the end, no cycle exists.

Time Complexity: O(n) Space Complexity: O(1)

FLOYD'S ALGORITHM
slow = fast = head
while fast and fast.next:
slow = slow.next
fast = fast.next.next
if slow == fast:
return True # Cycle Found
return False
