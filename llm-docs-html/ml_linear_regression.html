Linear Regression

Predicting continuous values using best-fit lines
Linear Regression
Type
Supervised (Regression)
Goal
Minimize Sum of Squared Residuals
Parameters
Slope (w), Intercept (b)

1. Summary

Linear Regression is the simplest algorithm in machine learning. It assumes a linear relationship between the input variable (X) and the output variable (y). The goal is to find the line of "best fit" that minimizes the vertical distance between the actual data points and the predicted line.

2. Visual Flow

graph TD A[Input Features X] --> B[Multiply by Weights w] B --> C[Add Bias b] C --> D[Predict Output y_hat] D --> E[Calculate Loss - MSE] E --> F[Optimize w and b via Gradient Descent] F --> B

3. Mathematics

y = wX + b

Loss Function (Mean Squared Error):

MSE = (1/n) * Σ(y_actual - y_predicted)²

5. Real-World Case Study: Real Estate Valuation

The Problem: A property portal (like Zillow) needs to estimate the market price of 50 million homes daily.

The Solution: Multiple Linear Regression calculates price based on features like:

  • Square Footage (Size)
  • Number of Bedrooms/Bathrooms
  • Proximity to Schools (Location Index)

Why Linear Regression? Its Coefficients are highly interpretable. If the coefficient for "Square Feet" is 2.5, the company can explicitly state: "Every extra square foot adds approximately $250 to the home value."

6. Implementation

👍 When to use

  • Predicting house prices or salaries (continuous values).
  • Establishing relationships between variables (e.g., ad spend vs. sales).
  • Baseline model - always start here.

👎 Limitations

  • Assumes linearity - fails on non-linear data (like XOR).
  • Sensitive to outliers (they pull the line significantly).
  • Assumes independence of features.

6. Implementation

from sklearn.linear_model import LinearRegression
import numpy as np

# 1. Prepare Data
X = np.array([[1], [2], [3], [4]])
y = np.array([2, 4, 6, 8])

# 2. Train Model
model = LinearRegression()
model.fit(X, y)

# 3. Predict
prediction = model.predict([[5]])
print(f"Prediction for 5: {prediction[0]}") # Output: 10.0
