1. Level Order Traversal (Breadth-First Search)

Intuition

Tree traversal usually follows a recursive, vertical depth-first pattern. However, certain problems (like finding the "view" of a tree or the shortest path in a graph-like tree) require Horizontal processing.

The Queue Strategy: BFS uses a First-In-First-Out (FIFO) Queue to maintain the "Discovery Order".

  • Start by pushing the Root into the Queue.
  • While the Queue is not empty, dequeue a node, "visit" it, and immediately enqueue its Left and Right children.
  • Because of the FIFO property, all nodes at distance d are processed before any node at distance d+1.
Time Complexity: O(n) Space Complexity: O(w) (Max Width)
